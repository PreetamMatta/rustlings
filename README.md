# Rust Learning Playground

This repository is a personal playground where I learn and practice the **Rust** programming language.

## Learning Resources

To guide my learning journey, I am using the following resources:

* **Book**: [The Brown University Rust Book](https://rust-book.cs.brown.edu/) - An excellent resource for deep dives into Rust concepts.
* **Tooling**: [Rustlings](https://rustlings.rust-lang.org/) - A collection of small exercises to get comfortable with Rust syntax and common patterns through hands-on practice.

Happy coding! 🦀
